Getting Started

Getting started with jest is pretty simple. If you want to test the following sum.js file,

// sum.js function sum(value1, value2) { return value1 + value2; } module.exports = sum;

1) Create a directory __tests__/ with a file sum-test.js

// __tests__/sum-test.js jest.dontMock('../sum'); describe('sum', function() { it('adds 1 + 1 to equal 2', function() { var sum = require('../sum'); expect(sum(1, 2)).toBe(3); }); });

2) Run npm install jest-cli --save-dev

3) Add the following to your package.json

{ ... "scripts": { "test": "jest" } ... }

4) Run npm test

[PASS] __tests__/sum-test.js (0.015s)
